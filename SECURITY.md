# Política de segurança

## Versões suportadas

Por se tratar de um projeto em estágio inicial, apenas a versão mais recente da
branch `main` recebe correções de segurança.

| Versão | Suportada |
| ------ | --------- |
| 0.1.x  | ✅        |

## Como reportar uma vulnerabilidade

**Não** abra _issues_ públicas para vulnerabilidades de segurança.

Em vez disso, envie um e-mail para **security@lumina.finance** com:

- uma descrição do problema e do impacto potencial;
- passos para reproduzir;
- versões/ambiente afetados.

Você receberá uma confirmação em até 72 horas e atualizações conforme a
investigação avança. Pedimos divulgação responsável: dê-nos tempo razoável para
corrigir antes de qualquer publicação.
