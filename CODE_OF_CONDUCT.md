# Código de Conduta

## Nosso compromisso

No interesse de promover um ambiente aberto e acolhedor, nós, como
contribuidores e mantenedores, comprometemo-nos a tornar a participação no nosso
projeto uma experiência livre de assédio para todos, independentemente de idade,
identidade de gênero, deficiência, etnia, nível de experiência, nacionalidade,
aparência, religião ou orientação sexual.

## Nossos padrões

Exemplos de comportamento que contribuem para um ambiente positivo:

- Usar linguagem acolhedora e inclusiva.
- Respeitar diferentes pontos de vista e experiências.
- Aceitar críticas construtivas com elegância.
- Focar no que é melhor para a comunidade.

Comportamentos inaceitáveis incluem assédio, comentários depreciativos e
ataques pessoais ou políticos.

## Aplicação

Casos de comportamento abusivo podem ser reportados para **conduct@lumina.finance**.
Todas as queixas serão analisadas e investigadas de forma justa e confidencial.

Este código é adaptado do [Contributor Covenant](https://www.contributor-covenant.org),
versão 2.1.
